linreg

Clara Schartner and Araya Eamrurksiri

2015-09-22

The package linreg includes a function linreg_function for linear regression.

Background

The linear model: \(Y = X\beta + \epsilon\)

Functions

By using the function linreg a linear model is calculated.

lin.result<-linreg_function(waiting~eruptions,faithful)
lin.result
## Call:
## linreg_function(formula = waiting ~ eruptions, data = faithful)
## 
## Coefficients:
## (Intercept)   eruptions 
##    33.47440    10.72964

In the output the results of the coeffients is shown together with the input formula.

residuals<-resid(lin.result)
head(residuals)
## [1]  6.898894  1.212248  4.763708  4.029832  2.888139 -9.407953

In order to obtain more information about this linear regression, the results of the linear regression are saved. Then there are a couple of functions which help to learn more about the results of the regression. The function predict returns all the predicted values.

predict(lin.result)

The function coef returns only the coefficients, which can be useful if you need to coefficients for further calculations.

coef(lin.result)
## (Intercept)   eruptions 
##    33.47440    10.72964

Using summary is very useful to get more information about the fit of the linear regression. You get the coefficients together with the standard errors, the t-value and the exact p-value. The degrees of freedom are also gotten from the output.

summary(lin.result)
## Call:
## linreg_function(formula = waiting ~ eruptions, data = faithful)
## 
## Coefficients:
##             Estimate Std.Error  t.value       p.value
## (Intercept) 33.47440 1.1548735 28.98534  7.136015e-85
## eruptions   10.72964 0.3147534 34.08904 8.129959e-100
## 
## Residual standard error:5.914009 on 270 degree of freedom

If you plot the results of the linear regression you get two plots: One showing the fitted values vs. the residuals together with a lowess of fit curve. The other plot shows the fitted values vs. the root of the absolute value of the standardized residuals.

plot(lin.result)
